# bootstrap + jquery + web3 + express 



## 目录结构

``` bash
demo
├── common/                 发送请求的success和fail的封装
├── index.html              主入口
├── package.json            npm包管理，脚本运行
├── node_modules/           包依赖安装的地方
└── test.js/                node开启服务
```

## 环境
``` bash
node -v

node v7.10.1 (npm v4.2.0)

```

## npm 相关命令 

``` bash
# 安装工程依赖
npm install

# node开启服务
npm run start

```
